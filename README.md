# Exercício de HTML e CSS

## Sobre o Projeto

Este projeto foi desenvolvido como parte dos meus estudos de HTML e CSS. O objetivo é praticar conceitos básicos de desenvolvimento web, colocando em prática a criação de estruturas de páginas e a aplicação de estilos utilizando CSS.

## Tecnologias Utilizadas

- HTML5
- CSS3

## Objetivos

- Aprimorar conhecimentos em HTML e CSS.
- Praticar a criação de layouts simples.
- Desenvolver páginas web utilizando boas práticas básicas.
- Entender melhor a organização e estilização de elementos.

## Conteúdo Praticado

Durante o desenvolvimento deste projeto foram utilizados conceitos como:

- Estrutura básica de uma página HTML.
- Títulos, textos e imagens.
- Divs e organização de conteúdo.
- Classes e identificadores.
- Estilização com CSS.
- Cores, espaçamentos e alinhamentos.
- Posicionamento de elementos.

## Finalidade

Este projeto possui caráter educacional e foi criado exclusivamente para fins de aprendizado e prática de programação front-end.
